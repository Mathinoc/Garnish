<h2><b>INTRODUCTION</b></h2>
<p class="">Gathering information across the web is web scraping, also known as <strong>Web Data Extraction &amp; Web Harvesting. </strong>Nowadays data is like oxygen for startups &amp; freelancers who want to start a business or a project in any domain. Suppose you want to find the price of a product on an eCommerce website. It&#8217;s easy to find but now let&#8217;s say you have to do this exercise for thousands of products across multiple eCommerce websites. Doing it manually; not a good option at all.<b><br />
</b></p>
<p>&nbsp;</p>
<h2><b>Get to know the Tool</b></h2>
<p class=""><b>JavaScript</b> is a popular programming language and it runs in any web browser.</p>
<p class=""><b>Node JS</b> is an interpreter and provides an environment for JavaScript with some specific useful libraries.</p>
<blockquote><p><b>In short, Node JS adds several functionality &amp; features to JavaScript in terms of libraries &amp; make it more powerful.</b></p></blockquote>
<div class="medium-insert-images">
<figure></figure>
<figure><img class="alignnone" src="https://editor.analyticsvidhya.com/uploads/50042Node Js libraries copy.jpg" alt="web scraping node js" width="920" height="604" /></figure>
</div>
<h6><b> </b></h6>
<h2><b>Hands-On-Session</b></h2>
<p class="">Let&#8217;s get to understand web scraping using Node JS with an example. Suppose you want to analyze the price fluctuations of some products on an eCommerce website. Now, you have to list out all the possible factors of the cause &amp; cross-check it with each product. Similarly, when you want to scrape data, then you have to list out parent HTML tags &amp; check respective child HTML tag to extract the data by repeating this activity.<b><br />
</b></p>

<li>Include the HTML tags in our Code</li>
<li>Cross-check the Scraped Data</li>
</ul>
